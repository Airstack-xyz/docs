# FollowFilter

## Example

```json
{
  "followerProfileId": 1,
  "followingProfileId": 2
}
```

## Params

| Name                 | Type  | Description                                         |
| -------------------- | ----- | --------------------------------------------------- |
| `followerProfileId`  | `Int` | The FID of the user that initiate follows.          |
| `followingProfileId` | `Int` | The FID of the user that is targeted for following. |
